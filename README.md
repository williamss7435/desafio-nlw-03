## Instalação
### Back-end 
```
cd happy-server
npm install
npm run typeorm migration:run
npm run dev
```

###Fron-end (Web)
```
cd happy-web
npm install
npm run start
```

###Fron-end (Mobile) `Requer o Expo instalado`
```
cd happy-mobile
npm install
npm run start
```
